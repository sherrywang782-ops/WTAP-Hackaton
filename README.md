# MentorMatch - Mentor-Mentee Connection Platform

A quirky, fun, and personal platform for connecting mentors with mentees. Built with a focus on meaningful relationships beyond just professional connections.

## Features

- **Role Selection**: Choose between Mentor, Mentee, or Planner roles
- **Quirky Design**: Purple-themed with fun animations and engaging UI
- **Personal Focus**: Emphasis on building personal connections, not just professional ones

## Getting Started

Simply open `index.html` in your web browser to view the landing page.

## Project Structure

```
├── index.html      # Main landing page
├── styles.css      # Quirky purple-themed styling
├── script.js       # Interactive functionality
└── README.md       # This file
```

## Design Theme

- **Primary Color**: Purple (#8B5CF6)
- **Accent Colors**: Pink, Blue, Yellow, Green
- **Style**: Quirky, fun, engaging with smooth animations
